Cloudflare Proxy Setup

Required for Runway photo upscaling. Free — 100k requests/day. ~15 min, no CLI needed.

  1. Free account at cloudflare.com
  2. Workers & Pages → Create Worker
  3. Paste the copied Worker script
  4. Add RUNWAY_API_KEY as Environment Variable
  5. Click Deploy
  6. Paste workers.dev URL → Runway Settings → Proxy URL
